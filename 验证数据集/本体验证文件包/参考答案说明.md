**TODO-2 · 多跳穿透 + 方向 + 防环**（`graph_traverse`，关2）
```python
paths = []
stack = [(startId, [startId])]
while stack:
    node, path = stack.pop()
    if len(path) - 1 >= hops:
        continue
    for nb in neighbors(node):
        if nb in path:        # 防环
            continue
        np = path + [nb]
        paths.append(np)
        stack.append((nb, np))
return {"start": startId, "rel": rel, "hops": hops, "direction": direction, "paths": paths}
```
要点：DFS/BFS 到 hops 跳；`path` 内去重防环；neighbors 按 direction 取 SH_OUT/SH_IN。
